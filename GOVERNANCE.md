# Governance

Kris Wang is the founding primary maintainer.

The primary maintainer is responsible for:

- project scope and roadmap;
- methodological review;
- issue triage and pull-request review;
- release approval;
- security response;
- documentation and contributor standards.

As sustained contributors emerge, the project may appoint core maintainers through a public governance update. Maintainer status will reflect demonstrated review, release, and community responsibilities.
